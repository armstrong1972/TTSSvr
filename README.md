# TTS Server
A simple TTS service using the kokoro model.
It listens on UDP port 11316, and after receiving JSON data, it plays the text content.
For the data format and usage, refer to UDPTest.py.

Please download the kokoro model files from the following URL and unzip it into the "models" subdirectory under the compiled exe file directory:
https://www.artcubevr.com/dw/kokoro.zip

This TTS voice playback service was developed in conjunction with the AI-2Bots project. Please follow my repository AI-2Bots.
